#FANTA-HUB
Progetto finale del percorso formativo "Master Full-Time Full-Stack Developer" della durata di 6 mesi.
E' un'applicazione tramite cui è possibile organizzare e svolgere le aste di fantacalcio con i propri amici.
Nella navbar sono presenti le sezioni Asta, Strategia, Notizie, Profilo.
Dopo aver effettuato l'accesso è possibile accedere alla sezione Asta in cui dobbiamo stabilire le regole dell'asta(budget partecipanti e nome), una volta creata l'asta dobbiamo invitare i nostri amici tramite l'apposito bottone.
Dopo aver invitato i nostri amici si creerà una connessione tramite il protocollo WebSocket che consentirà la comununicazione in tempo reale per avere le offerte tutti allo stesso momento, in più è possibile comunicare con loro tramite l'apposita chat.
La sezione strategia ci permette di creare una strategia personale che sarà poi esportabile come file CSV.
La sezione notizie è una pagina di informazione in cui sono presenti le ultime notizie del mondo del calcio italiano grazie al collegamento tramite tecnologia RSS con la Gazzetta Dello Sport e alcuni iframe di SofaScore.
La sezione profilo permette di modificare l'immagine e tutti i dati dell'utente loggato.
Front End sviluppato con React e Bootstrap 
Back End con Sprin Boot e database con PostgreSQL
Nella homepage sono presenti alcuni elementi animati della libreria ReactBits e bottoni per effetturare accesso o registrazione.
